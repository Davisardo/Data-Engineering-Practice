Course 02 - Practical Python for Data Work (IBM Data Engineering Professional Certificate)

## Tujuan Lab Ini

- Menggunakan ekspresi dan variabel untuk melakukan operasi.

## Struktur Folder

1.3 - Expressions and Variables/
└── expressions_variables.py

## Materi yang Dipelajari

- Operasi aritmatika dasar: `+`, `-`, `*`, `/`, `//` (integer division).
- Urutan operasi matematika (operator precedence) — perkalian/pembagian didahulukan dari penambahan/pengurangan, tanda kurung diproses lebih dulu.
- Menyimpan hasil ekspresi ke variabel, dan menimpa (overwrite) nilai variabel yang sudah ada.
- Penamaan variabel yang bermakna (`total_min`, `total_hours`) untuk keterbacaan kode.

## Cara Menjalankan

python expressions_variables.py

## Catatan Pribadi

Perbedaan penting: `/` selalu menghasilkan `float` meski hasilnya bulat, sementara `//` membulatkan ke bawah dan menghasilkan `int` (kecuali salah satu operand float, tetap jadi float dengan `.0`).
